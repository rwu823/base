## Infrastructure

Everything you need. it includes:

* ESLint
* TSLint
* tsconfig
* prettier with precommit hook integration
* babel-node / ts-node

## Installation

```sh
npm i -D https://github.com/rwu823/base
```

## Initialization

```sh
npx base-init
```
