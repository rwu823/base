## Infrastructure

Everything you need. it includes:

* ESLint
* TSLint
* tsconfig
* prettier with precommit hook integration
* babel-node / ts-node

## Installation

```sh
npm i -D github:rwu823/base
```

## Initialization

```sh
npx base-init
```

## Version Convention
base's major version base on `Babel`'s version then suffix YY.M.D. So it looks like `7.0.0-18.8.9`.
